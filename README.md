# HomeAssistant JVC Projector Integration

This is a [HACS](https://www.hacs.xyz/) version of the official [JVC Projector](https://www.home-assistant.io/integrations/jvc_projector/) integration used to test early beta releases.

## Installation

1. You must remove the official JVC Projector integration if you have it installed currently. You can reinstall it after testing and removing this HACS version.
2. Install HACS into HomeAssistant if you don't already have it. See https://www.hacs.xyz/docs/use/download/download/.
3. Let HACS know about this custom integration by adding the repo https://github.com/SteveEasley/homeassistant_jvc_projector as a custom repository. See https://hacs.xyz/docs/faq/custom_repositories.
4. Restart HomeAssistant.
5. In HomeAssistant go back to HACS and download the JVC Projector integration locally by searching for it and clicking Download.
6. And finally add the integration to HomeAssistant by going to Settings | Devices and Services and searching for it.
